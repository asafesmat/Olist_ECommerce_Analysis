# Olist E-Commerce Analysis Dashboard

## 📊 Project Overview
Comprehensive data analysis of Olist Brazilian E-Commerce dataset using SQL, Python, and Power BI. 
This project analyzes sales performance, delivery efficiency, seller metrics, and customer satisfaction.

## 🎯 Key Metrics

### Sales & Revenue
- **Total Revenue:** R$ 14.02M
- **Total Orders:** 113.314K
- **Average Order Value:** R$ 123.45
- **Top Category:** beleza_saude

### Delivery Performance
- **Average Delivery Time:** 12.02 days
- **On-Time Delivery Rate:** 92.4%
- **Late Delivery Rate:** 7.6%

### Customer Satisfaction
- **Average Review Score:** 4.03/5.00
- **Total Reviews:** 96.34K
- **5-Star Reviews:** 63%

### Seller Performance
- **Total Sellers:** 1,766
- **Average Seller Rating:** 4.03⭐
- **Top Seller Revenue:** R$ 226,987.93

## 📊 Dashboard Pages

### Page 1: Executive Summary
KPI cards, delivery status, monthly trends, top categories, seller distribution

### Page 2: Sales & Revenue Analysis
Monthly revenue trend, category performance, payment methods, revenue by state

### Page 3: Delivery Performance
Late delivery rates by state, delivery days distribution, performance metrics

### Page 4: Customer Satisfaction
Review score distribution, delivery impact analysis, category ratings

## 🛠️ Tools & Technologies

- **Database:** SQL Server (SSMS)
- **Data Processing:** Python (Pandas, NumPy)
- **Visualization:** Power BI Desktop
- **Version Control:** Git & GitHub

## 📈 Key Insights

1. **Strong On-Time Performance:** 92.4% of orders delivered on time
2. **Revenue Decline:** Downward trend from 2016-2018, indicating market saturation
3. **São Paulo Dominance:** SP state accounts for majority of revenue and orders
4. **Payment Preference:** Boleto is most common payment method (76.81%)
5. **Excellent Customer Satisfaction:** 4.03/5 average rating with 63% 5-star reviews
6. **State Variation:** AL state has highest late delivery rate (23.93%)

### Data Pipeline
1. Raw CSV files loaded into SQL Server
2. SQL queries extract and aggregate data
3. Python notebook performs EDA and validation
4. Processed data imported to Power BI
5. Interactive dashboard created with 4 analysis pages
